Расчет прибыли (убытка) по валютному активу.
При нажатии на кнопку recalculate, приложение скачивает курс обмена Доллара США на рубли из сервиса https://openexchangerates.org (см. https://docs.openexchangerates.org/docs/) на дату покупки и на текущую.
Далее производится расчет прибыли или убытока если бы валюту продали сегодня по текущему курсу. При расчете помимо разницы курсов учитывается и спред (spread), равный 0.5% Прибыль выводится в рублях в нижнем правом углу.
